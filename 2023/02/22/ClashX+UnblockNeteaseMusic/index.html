ClashX Pro + UnblockNeteaseMusic

发表于 2023-02-22 更新于 2024-02-23 阅读次数：

准备

  • UnblockNeteaseMusic/server

  • ClashX Pro

  • 网易云音乐 2.3.12_956

  • 安装证书 ca.crt

ClashX Pro 设置并添加规则

  • 开启 Enhanced mode

    • Enhanced mode Config → DNS mode → Mapping

  • 编辑 config.yaml

    proxies:
  - { name: NeteaseMusicUnlock, type: http, server: 192.168.1.2, port: 8080 }
    proxy-groups:
  - name: NeteaseMusic
type: select
proxies:
  - DIRECT
  - NeteaseMusicUnlock
    rules:
  - DOMAIN-SUFFIX,music.163.com,NeteaseMusic
  - DOMAIN-SUFFIX,music.126.net,NeteaseMusic
