VMware-tools 无法安装问题解决

发表于 2023-02-06 更新于 2024-02-23

问题描述

  • WMware版本 WMware Workstation PRO 16.2.3
  • 安装镜像版本:Windows7 x64
  • 自动安装的VMware Tools 版本为11.3.5
    出现提示Windows无法验证此驱动程序软件发布者
    导致出错无法安装

图示如下



解决办法

  • 安装 VMware Tools 11.0.0
  • 挂载iso镜像到虚拟机
  • 运行setup64.exe
  • 安装成功后重启
